Font assets
===========

The site self-hosts these open fonts so the public page matches the selected MCC specimens:

- AdwaitaSans-Regular.ttf: Adwaita Fonts, GNOME, SIL Open Font License 1.1.
- AdwaitaMono-Bold.ttf: Adwaita Fonts, GNOME, SIL Open Font License 1.1.
- LiberationMono-Bold.ttf: Liberation Fonts, Red Hat, SIL Open Font License 1.1.

Adwaita Fonts source: https://gitlab.gnome.org/GNOME/adwaita-fonts
Liberation Fonts source: https://github.com/liberationfonts/liberation-fonts
